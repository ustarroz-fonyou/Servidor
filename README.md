Servidor hecho con NodeJs, Express y Socket.io



## Installation

Este servidor requiere [Node.js](https://nodejs.org/) v10+ para funcionar

Clonar el repositorio en tu maquina local

```sh
cd servidor
npm i
```

Para arrancar el servidor en terminal usar

```sh
npm run dev
```
